# 💸 Expense Flow

A comprehensive full-stack financial tracking web application designed to help users monitor spending habits, manage outstanding liabilities, and organize cash flow with a streamlined interface.

---

## 🚀 Key Features

### 🔐 User Authentication

Expense Flow includes a secure authentication system to protect user data and ensure privacy.

### 📊 Expense Management

Effortlessly add, track, and review daily expenditures with a structured historical log.

### 💳 Debt Tracking

Dedicated modules to record and monitor debts, ensuring clear visibility of lending and borrowing.

### 🏷️ Dynamic Categorization

Use the **"Add Tag"** feature to organize transactions into custom categories like:

* Food 🍔
* Travel ✈️
* Bills 📄
* Personal 💼

### 🔐 Secure Data Storage

Reliable and structured storage powered by **MySQL integration**.

### 📱 Responsive UI

Clean, mobile-friendly interface built with modern frontend technologies for seamless use across devices.
